# 🚀 Deploy Rápido - 3 Passos

## Opção Mais Fácil: Netlify

### 1️⃣ Criar Conta GitHub
- Acesse: https://github.com/signup
- Crie sua conta gratuita

### 2️⃣ Upload do Código
1. Acesse: https://github.com/new
2. Nome: `monitoramento-interdicoes`
3. Clique em "Create repository"
4. Clique em "uploading an existing file"
5. Arraste TODOS os arquivos do projeto
6. Clique em "Commit changes"

### 3️⃣ Deploy no Netlify
1. Acesse: https://app.netlify.com/signup
2. Clique em "Sign up with GitHub"
3. Clique em "Add new site" → "Import an existing project"
4. Escolha "GitHub"
5. Selecione `monitoramento-interdicoes`
6. Clique em "Deploy site"
7. **Pronto!** Seu site estará online em 2-3 minutos

### 🌐 Acessar
Sua URL será algo como: `https://random-name.netlify.app`

Você pode personalizar em "Site settings" → "Change site name"

## 🎯 Alternativas

### Vercel
1. https://vercel.com/signup
2. Conecte com GitHub
3. Selecione o repositório
4. Deploy automático

### GitHub Pages
1. No repositório, vá em "Settings" → "Pages"
2. Em "Source", selecione "GitHub Actions"
3. Commit na branch `main`
4. Deploy automático
